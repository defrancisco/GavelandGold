# Gavel & Gold - Sistema de Subastas

[cite_start]Repositorio oficial de la aplicación móvil "Gavel & Gold", desarrollada para el proyecto "Sistema de Subastas" [cite: 2] [cite_start]correspondiente a la materia **Desarrollo de Aplicaciones I (Primer Cuatrimestre 2026)**.

## Descripción del Proyecto
[cite_start]Gavel & Gold es una aplicación móvil diseñada para que los usuarios puedan participar de forma on-line en las subastas presenciales de la empresa[cite: 4]. [cite_start]La plataform permite a los usuarios registrados actuar como oferentes pujando por artículos en tiempo real, así como también solicitar la evaluación y posterior inclusión de bienes propios en futuras subastas[cite: 3, 4]. [cite_start]La app está pensada para integrarse y sincronizar datos con el sistema local existente de la empresa[cite: 5].

## Tecnologías Utilizadas
* **Frontend Móvil:** Android Nativo (Kotlin / XML) desarrollado en Android Studio.
* **Diseño y Prototipado:** Figma.
* **Documentación de API:** Swagger.

## Instalación y Ejecución (Modo Desarrollo)

1. Clonar este repositorio:
   `git clone [URL_DEL_REPOSITORIO]`
2. Abrir el proyecto utilizando **Android Studio**.
3. Esperar a que finalice la sincronización de Gradle.
4. Ejecutar la aplicación (`Run 'app'`) utilizando un emulador con API 24 o superior, o un dispositivo físico conectado.
